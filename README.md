## weather app

ℹ️ Sobre o projeto

### Projeto usando React com useContext, e useReducer (similar ao redux), usando geoLocalização, e a api do openWeather , com dinamico background se a temperatura estiver menor que 16 graus ou maior .

✨ Demonstração
